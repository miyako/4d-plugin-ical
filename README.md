4d-plugin-ical
==============

4D plugin to access to the CalendarStore API (10.5 to 10.7). For 10.8 and Reminders, use Event Kit.

##Update

A **64 bits** version is available on the [v2](https://github.com/miyako/4d-plugin-ical/tree/v2) branch.
